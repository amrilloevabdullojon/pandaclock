## Компоненты

### Верхний контрол
- **Кнопка "Сегодня"** — быстрый возврат
- **Стрелки навигации** + название периода (Май 2026)
- **Toggle вида:** Месяц / Неделя / День
- **Кнопка "+ Создать"** — открывает выбор типа события

### Фильтры
- **Сотрудник** — multi-select (для руководителя/HR)
- **Отдел** — multi-select
- **Тип события** — multi-select (отпуска / больничные / праздники / митинги / дедлайны)

### Ячейка дня
- Число дня крупно
- Иконки событий (max 3 видимых, остальные "+N")
- Выходные — серый фон
- Сегодня — фиолетовая рамка
- Hover: легкая подсветка

### Клик на день
Открывается popover с детальным списком событий + кнопка "+ Добавить в этот день".

## Компоненты

### Список чатов (левая колонка)
- Поиск чатов и сообщений (debounced 300ms)
- **Каналы** — групповые чаты (отделы + кастомные)
  - Префикс `#`
  - Бейдж с числом непрочитанных (`primary-500` фон)
- **Личные** — direct messages
  - Аватар + имя
- Кнопки внизу: "+ Новый чат", "+ Создать канал"

### Окно чата (правая колонка)
- **Хедер:** название чата + кнопка видеозвонка + меню (⋮ — добавить участников, заглушить, выйти)
- **Лента сообщений** — обратный порядок (новые внизу), auto-scroll
- **Группировка** — несколько сообщений от одного автора подряд показываются без повторения аватара/имени
- **Разделители даты** — "▔▔▔ Сегодня ▔▔▔", "▔▔▔ Вчера ▔▔▔", "▔▔▔ 22 мая ▔▔▔"
- **Реакции на сообщения** — выпадают при hover

### Сообщение

```
[👤] Анна Каримова  •  10:25
Готов на 80%, к вечеру отправлю
👍 3   ❤️ 1   [+ реакция]
```

При hover:
```
                                        [👍 ↩️ 📌 ⋮]
[👤] Анна Каримова  •  10:25
Готов на 80%, к вечеру отправлю
```

### Input
- Однострочный с авто-расширением до 4 строк
- Поддержка markdown (жирный, курсив, код)
- Эмодзи-пикер
- Прикрепление файлов (drag-n-drop, paste из буфера)
- Кнопка отправки (или Enter)
- `@упоминания` — autocomplete сотрудников
- `#упоминания` — autocomplete каналов

### Видеозвонок (кнопка в хедере)
- Открывает оверлей с LiveKit
- Все участники чата уведомляются (push + красная плашка в чате)

## Mobile-версия

3-уровневая навигация:
1. **Список чатов** (главный экран таба "Чаты")
2. **Окно чата** при тапе
3. **Детали чата** (участники, файлы) — свайп влево

Input закреплён внизу, над клавиатурой.

## Особенности

### Экспорт
- **Excel:** все данные с фильтрами, форматирование, готовые формулы
- **PDF:** официальный документ с шапкой компании, логотипом, подписью
- **Email:** отправка на указанный email с PDF во вложении
- **Async:** генерация в очереди (BullMQ), пользователю показывается прогресс

### Расписание отчётов
В тарифе Pro: возможность настроить **автоматическую отправку** отчётов раз в неделю/месяц на email руководителя.

### Кастомные отчёты (Этап 3)
Возможность создать свой отчёт с выбором метрик и группировок.

## Вкладки

### 4.1 Профиль компании
(см. выше)

### 4.2 Рабочее время
- Стандартный рабочий график (например, 9:00-18:00, Пн-Пт)
- Длительность обеденного перерыва
- "Что считать опозданием" (e.g., +15 минут)
- Праздничные дни (календарь Узбекистана + кастомные)
- Минимальная/максимальная норма часов в день

### 4.3 Политики
- Геолокация при отметке (отключена / радиус / мульти-точки)
- Контроль IP (whitelist для офисов)
- Срок действия паролей
- Политика отпусков (количество дней в год, как накапливаются)
- Многоуровневое согласование заявок (выкл / 1 уровень / 2 уровня)

### 4.4 Уведомления
- Настройки канала: email / push / Telegram / SMS
- Какие события отправлять (чеклист)
- Время "не беспокоить" (19:00-9:00 по умолчанию)

### 4.5 Безопасность
- Обязательное 2FA для всех / для админов / опционально
- Минимальная длина пароля
- Сложность пароля
- Audit log (просмотр + экспорт)
- Активные сессии (с возможностью отозвать)

### 4.6 Интеграции
- Google Workspace (SSO) — кнопка "Подключить"
- Microsoft 365 (SSO)
- Slack (notifications)
- Telegram (notifications + бот)
- 1С (Этап 3)

## Особенности

### Уведомления о биллинге
- Email + push за 7 дней до списания
- Email при успешной оплате с чеком (PDF)
- Email при неудачной оплате + 3 повторных попытки в течение 7 дней
- После 7 дней неуплаты — блокировка функций (мягкая, только для администраторов)
- После 14 дней — режим read-only (нельзя добавлять/менять)
- После 30 дней — полная блокировка (можно экспортировать данные)

### Триал
- 14 дней с момента регистрации
- За 3 дня до конца — баннер в интерфейсе
- За 1 день — email + push
- После окончания — экран "Триал закончился. Выберите тариф"

### Документы для бухгалтерии
- Автоматические счета-фактуры (для юрлиц)
- Акты выполненных работ
- ЭЦП (Этап 4) — для подписания актов
- Все документы доступны для скачивания в истории платежей
