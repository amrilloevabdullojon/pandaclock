## 2. Типографика

### Шрифт: Nunito (как в шаблоне)
- **Бесплатный** (Google Fonts)
- **Поддержка кириллицы** ✅
- **Поддержка узбекской латиницы** ✅
- Дружелюбный, читаемый, B2B-friendly

**Установка в Next.js:**
```typescript
import { Nunito } from "next/font/google";

const nunito = Nunito({
  subsets: ["latin", "cyrillic"],
  weight: ["400", "600", "700"],
  variable: "--font-nunito",
});
```

### Шкала размеров (web)

```
display-1   60px / 64px / 800   — Hero на landing
display-2   48px / 56px / 800   — Hero internal pages

h1          36px / 44px / 700   — Заголовок страницы (Overview)
h2          28px / 36px / 700   — Секции
h3          22px / 30px / 600   — Карточки, подсекции
h4          18px / 26px / 600   — Группы полей
h5          16px / 22px / 600
h6          14px / 20px / 600

body-lg     18px / 28px / 400   — Lead-параграф
body        16px / 24px / 400   — Основной текст
body-sm     14px / 20px / 400   — Вторичный текст
caption     12px / 16px / 400   — Подписи, метки
overline    11px / 16px / 600 (uppercase, letter-spacing 0.5) — Маркеры

button-lg   16px / 24px / 600
button      14px / 20px / 600
button-sm   13px / 18px / 600

number-xl   48px / 56px / 800   — KPI в дашбордах (как "25.1k")
number-lg   32px / 40px / 700   — Числа в виджетах
number-md   24px / 32px / 700
```

### Шкала размеров (mobile)
Уменьшаем display и h1, остальное похоже:
```
h1-mobile   28px / 36px / 700
h2-mobile   22px / 30px / 700
button большая (clock-in) — 18px / 24px / 700
```

## 3. Радиусы (corner radius)

```
radius-xs   4px   — мелкие бейджи
radius-sm   6px   — инпуты, маленькие кнопки
radius-md   12px  — стандартные карточки задач
radius-lg   16px  — крупные карточки (KPI, dashboard widgets)
radius-xl   24px  — highlight-карточки
radius-2xl  32px  — модалки, hero-блоки
radius-pill 9999px — кнопки-таблетки, search bar, badges
radius-full 50%   — аватары, иконки в кругах
```

⭐ **Стиль шаблона:** large radius (16-24px) на всех ключевых элементах. Берём это правило.

## 5. Spacing (отступы)

Шаг 4px (Tailwind-совместимая):
```
0     0px
0.5   2px
1     4px
1.5   6px
2     8px
3     12px
4     16px   — стандартный gap между элементами
5     20px
6     24px   — padding карточек
8     32px   — секции
10    40px
12    48px
16    64px
20    80px   — большие промежутки между секциями
24    96px
32    128px
```

Применение:
- Padding карточек: `p-6` (24px)
- Gap между карточками: `gap-6` (24px) или `gap-4` (16px)
- Padding страницы: `px-8 py-6`
- Между секциями: `mb-8` или `mb-12`

## 7. Иконки

**Замена:** в шаблоне иконки разные/старые → используем **Lucide Icons** (идут с shadcn/ui).

**Стиль:**
- Outline (не filled)
- Толщина обводки: 2px
- Размеры: 16px (inline), 20px (стандарт), 24px (крупные)

**Основные иконки для HR:**
- Дашборд: `LayoutDashboard`
- Сотрудники: `Users`
- Время: `Clock`
- Задачи: `CheckSquare`
- График: `Calendar`
- Заявки: `FileText`
- Чаты: `MessageCircle`
- Отчёты: `BarChart3`
- Настройки: `Settings`
- Уведомления: `Bell`
- Поиск: `Search`
