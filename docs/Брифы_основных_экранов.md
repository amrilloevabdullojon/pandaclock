## Компоненты подробно

### TIME WIDGET (главный)

**Состояния:**
- **Не начали день:**
  - Большая кнопка `success-500`: "Начать день"
  - Под ней: "Сегодня вы ещё не отмечались"
- **На работе:**
  - Заголовок: "Вы на работе"
  - Большое число: текущее время с начала дня (обновляется live)
  - Доп. инфо: "Начали в 9:02 • IP/локация проверены ✓"
  - 2 кнопки: `[Завершить день]` (danger), `[Перерыв]` (secondary)
- **На перерыве:**
  - Заголовок: "Перерыв"
  - Большое число: время перерыва
  - Кнопка: `[Вернуться к работе]` (primary)
- **День завершён:**
  - Заголовок: "Хороший день! 👋"
  - Текст: "Отработано: 8ч 15м"
  - Disabled-кнопка: "Начать день" (доступна с завтра)

**Визуальный стиль:** карточка `radius-xl`, `shadow-sm`, фон белый, акцент primary.

### KPI CARDS (3 штуки)

Все по одному шаблону:
```
┌─────────────────────┐
│ [иконка]   [период] │
│                     │
│ ЗАГОЛОВОК           │ ← overline, neutral-500
│ БОЛЬШОЕ ЧИСЛО       │ ← number-lg, neutral-900
│ Описание / change   │ ← caption, success/danger
└─────────────────────┘
```

Карточки:
1. **Часы за неделю** — иконка `Clock`, период "за 7 дней"
2. **Задачи выполнено** — иконка `CheckSquare`, формат "X / Y"
3. **Опоздания** — иконка `AlertCircle`, "за месяц"

### TODAY'S TASKS

Список карточек задач (компактный вариант):
```
┌────────────────────────────────────────┬──────┐
│ [□] Подготовить отчёт                  │ 14:00│
│     Высокий приоритет • #Маркетинг      │      │
└────────────────────────────────────────┴──────┘
```

- Checkbox слева — быстрая смена статуса
- Текст задачи (1-2 строки максимум)
- Бейджи приоритета и проекта
- Время дедлайна справа
- Hover: подсветка фона `neutral-50`

Внизу: кнопка-ссылка `[+ Все задачи →]` ведёт на /tasks

### WEEK CHART

Bar chart за 7 дней:
- X: понедельник-воскресенье
- Y: часы (0-12)
- Цвет столбцов: `primary-500`
- Текущий день: `primary-700` (выделен)
- Над каждым столбцом — число часов мелким текстом

### TEAM ON WORK NOW

Список коллег с онлайн-статусом:
```
┌────────────────────────────────┐
│ [Avatar] Бахром Ю.   🟢 6ч 23м │
│ [Avatar] Дильноза А. 🟡 Перерыв│
│ [Avatar] Эльдар С.   🟢 4ч 15м │
└────────────────────────────────┘
```

- Зелёный кружок = на работе
- Жёлтый = перерыв
- Серый = офлайн
- Hover: подсказка "Начал день в 9:00"

### RECENT ACTIVITY

Текстовая лента действий (последние 5-7):
- Иконка типа события слева
- Текст в одну строку
- Время справа (мелкий шрифт, neutral-400)

## Компоненты подробно

### KPI ROW (4 карточки)

Точно как в Ventes-шаблоне:
1. **Сотрудников всего** — общее количество в отделе, +N новых за месяц
2. **На работе сейчас** — текущее количество + процент от общего
3. **Задач выполнено** — процент за период + change
4. **Заявок ждёт** — количество pending, кнопка "Решить"

### TOP PERFORMERS

Адаптация "Top Sales Representative" из шаблона:
- 5-10 лучших сотрудников по метрикам недели
- Аватар, имя, отдел, статус, метрика
- Бейдж "⭐ Top" у лидера
- Hover: переход к карточке сотрудника

**Toggle над списком:** `[На работе сейчас]` | `[Top performers]` — переключение между двумя видами.

### ATTENDANCE CHART

Адаптация Claims Over Years chart:
- Линейный график за месяц (по дням)
- 2 линии: "Пришли вовремя" (success) + "Опоздали" (danger)
- Tooltip при hover: точная цифра
- Период переключается (день/неделя/месяц/квартал)

### TODAY'S TARGET CARD (тёмная highlight)

Самый WOW-компонент шаблона:
- Тёмно-фиолетовый фон с волнистой текстурой
- Белый текст
- 2 секции внутри:
  - "Команда сегодня": число + аватары
  - "Задач выполнено": число + мини-график (sparkline)

### PENDING REQUESTS

Список заявок, ждущих решения:
- Аватар + имя + тип заявки + даты
- 2 кнопки: `[Утвердить]` (success), `[Отклонить]` (danger)
- При клике — модалка с подробностями

Под списком: `[Все заявки →]`

## Технические детали

### Геолокация
- При первом запуске: запрос разрешения с объяснением "зачем"
- Перед нажатием "Начать день": проверка нахождения в радиусе офиса
- Если вне зоны: попап "Вы не в зоне офиса. Подтвердить отметку?" + поле комментария

### Биометрия
- При входе в приложение — опциональный Face ID / Touch ID / Fingerprint
- Настраивается в Profile

### Offline-режим
- Если нет интернета — отметка сохраняется локально
- Иконка статуса в header: 🌐 (online) или 📴 (offline)
- При появлении интернета — авто-синхронизация + push "Отметка отправлена"

### Анимации
- Кнопка "Начать день" при нажатии: shrink + haptic feedback
- Переход между состояниями: cross-fade с количеством секунд
- Live-обновление времени "4ч 23м" — раз в минуту (не в секунду, чтобы не разряжать батарею)

## Tab Bar (нижняя навигация)

5 пунктов:
1. **🏠 Главная** — этот экран
2. **✅ Задачи** — список и канбан
3. **💬 Чаты** — отделы и личные
4. **📋 Заявки** — отпуска, больничные
5. **👤 Профиль** — статистика, настройки

Активный пункт: `primary-500` иконка + текст. Неактивный: `neutral-400`.

## Дополнительные элементы (общие для всех экранов)

### Header (всех мобильных экранов)
- Кнопка ☰ слева — открывает sidebar (для редких действий)
- Заголовок (опционально, может отсутствовать на главной)
- 🔔 — уведомления с badge
- 👤 — переход в профиль

### Empty states
Если у сотрудника пока нет задач/заявок/чатов:
- Иллюстрация маскота (см. Маскот_AI_промпты.md)
- Текст: "Здесь пока пусто"
- Кнопка действия: "Создать первую задачу" / "Написать в чат"

### Toast-уведомления
- Появляются снизу, исчезают через 4 секунды
- Цвет в зависимости от типа: success/warning/danger/info
- Содержат иконку + короткий текст
