## 📱 Установка

### Android

1. Скачай APK по ссылке (которую прислал отправитель этого письма)
2. Android спросит «Разрешить установку из неизвестных источников» — разреши
3. Установи, открой
4. Иконка — фиолетовая с пандой 🐼

### iOS

iOS-версия пока недоступна — Apple требует подтверждённый Developer
аккаунт ($99/год) для распространения. Если есть желание — пиши, попробуем
через TestFlight.

### Web

В браузере открой: **https://app.pandaclock.uz** _(когда домен подключим — пока временно `https://pandaclock-web-staging.vercel.app`)_

## 🐛 Куда писать баги

- В чат отправителю с темой **«[bug] что-то»**
- Чем подробнее тем лучше: что нажал → что ожидал → что произошло
- Скриншот/видео — золотой стандарт
- Если упало целиком — обычно достаточно «не работает на странице X»

### Что нас особенно интересует

- Тексты которые сбивают с толку
- Кнопки которые ведут в никуда
- Места где «вроде должно работать но не работает»
- Тормоза > 3 секунд
- Что-то откровенно уродливое визуально
