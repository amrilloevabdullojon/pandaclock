# Email-шаблоны
## Pandaclock — все системные письма

**Версия:** 1.0
**Дата:** 2026-05-24
**Отправитель:** `noreply@pandaclock.uz` (системные) и `team@pandaclock.uz` (от команды)

---

## Общие принципы

### Брендинг
- **Шапка:** логотип `🐼 Pandaclock` слева, маленький
- **Цвета:** primary `#5B4FE2`, текст `#1F2233`, фон `#FAFBFD`
- **Кнопка-CTA:** фиолетовая, скругление 12px, padding 16px 32px, белый текст
- **Подвал:** контакты + соц-сети + ссылка "Отписаться" (для маркетинговых, не для системных)

### Технические требования
- Responsive (mobile-first)
- Поддержка тёмной темы (`@media (prefers-color-scheme: dark)`)
- Alt-теги для всех изображений
- Plain-text версия каждого письма (для клиентов без HTML)
- Размер ≤ 100KB (иначе Gmail обрежет)

### Локализация
Каждое письмо в 3 версиях: русский, узбекский (латиница), английский. Локаль определяется языком пользователя.

### Сервис рассылки
- **MVP:** Resend (хороший API, дешёво)
- **Production:** Resend / SendGrid + собственный SMTP fallback

# Технические детали

## Структура HTML-шаблона

Использовать **MJML** (https://mjml.io/) — компилируется в кросс-браузерный HTML.

Базовый skeleton:
```mjml
<mjml>
  <mj-head>
    <mj-title>{{subject}}</mj-title>
    <mj-font name="Nunito" href="https://fonts.googleapis.com/css?family=Nunito" />
    <mj-attributes>
      <mj-all font-family="Nunito, sans-serif" />
      <mj-text color="#1F2233" line-height="1.5" />
      <mj-button background-color="#5B4FE2" border-radius="12px" />
    </mj-attributes>
  </mj-head>
  <mj-body background-color="#FAFBFD">
    <mj-section>
      <mj-column>
        <mj-image src="logo.png" alt="Pandaclock" width="140px" align="left" />
        <!-- Контент письма -->
      </mj-column>
    </mj-section>
    <mj-section>
      <mj-column>
        <mj-divider border-color="#E8EAF2" />
        <mj-text font-size="12px" color="#6B7080" align="center">
          🐼 Pandaclock — HR-система, которая работает за вас<br/>
          <a href="https://pandaclock.uz">pandaclock.uz</a>
        </mj-text>
      </mj-column>
    </mj-section>
  </mj-body>
</mjml>
```

## Тестирование
- **Litmus** или **Email on Acid** — проверка во всех почтовых клиентах
- Тестировать минимум в: Gmail (web, iOS, Android), Outlook (web, desktop), Apple Mail, Yandex.Mail

## SPF, DKIM, DMARC
Обязательно настроить, иначе письма попадут в спам:
- SPF-запись в DNS
- DKIM-подпись от Resend/SendGrid
- DMARC-политика (минимум `p=quarantine`)
